Comandos Linux

O Linux é um sistema poderoso usado em servidores, redes e desenvolvimento. Seus comandos permitem controle total do sistema operacional, desde manipular arquivos até monitorar processos e usuários.

Navegação

  • pwd — mostra o diretório atual.
  • ls — lista arquivos e pastas.
  • cd <pasta> — muda de diretório.
  • cd .. — volta um nível.

Manipulação de Arquivos

  • mkdir — cria uma pasta nova.
  • rm -r — remove pasta e conteúdo.
  • touch — cria arquivo vazio.
  • mv — move ou renomeia arquivos.

Permissões

  • chmod — altera permissões.
  • chown — muda o dono do arquivo.

Comandos Git

O Git é um sistema de controle de versão usado para gerenciar projetos e colaborar em equipe. Ele registra mudanças no código e permite voltar a versões anteriores ou trabalhar em branches paralelas.

Configuração

  • git config --global user.name — define seu nome.
  • git config --global user.email — define seu e-mail.

Controle de Versão

  • git init — inicia repositório.
  • git add . — adiciona arquivos.
  • git commit -m "msg" — salva mudanças.
  • git status — mostra o estado atual.

Conexão Remota

  • git push origin main — envia commits.
  • git pull origin main — atualiza do remoto.

Hospedagem Estática

Plataformas de hospedagem estática publicam sites feitos em HTML, CSS e JS diretamente a partir de um repositório Git. São ideais para portfólios, landing pages e projetos front-end.

Vercel

Especializada em frameworks modernos (Next.js, React). Faz deploy automático a cada push. Alta performance e CDN global.

Netlify

Simples e poderosa. Oferece SSL gratuito, funções serverless e integração com repositórios Git. Interface intuitiva.

Render

Hospeda sites estáticos e também backends. Boa escolha se o projeto crescer e precisar de APIs ou banco de dados.

DNS e HTTPS

DNS

Traduz nomes de domínio (como meusite.com) em endereços IP. Atua como “agenda da internet”.

  • A — aponta para IPv4.
  • AAAA — aponta para IPv6.
  • CNAME — redireciona para outro domínio.
  • NS — define os servidores de nome.

HTTPS

Protocolo seguro que criptografa a comunicação entre navegador e servidor. Garante privacidade e autenticidade.

  • Criptografia de dados.
  • Certificados SSL/TLS.
  • Requisito essencial para SEO e segurança.

Plataformas Git

Essas plataformas hospedam repositórios Git e oferecem recursos de colaboração e integração contínua.

GitHub

Maior comunidade open-source. Interface intuitiva e integração com diversas ferramentas.

GitLab

Voltado a equipes DevOps. Permite auto-hospedagem e pipelines completos de CI/CD.

Bitbucket

Focado em equipes corporativas. Integra-se com Jira e Confluence, da Atlassian.
